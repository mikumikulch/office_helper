# About

利信办公小助手是一个帮助你脱离繁琐的事务工作的自动化脚本应用。
小助手拥有诸如自动填写打印加班审批单、调休审批单、智能化打卡提示等功能。

# Version

**Version 2.0**  

- 支持自动填写加班审批表，并通过邮件发送到指定邮箱。
- 支持自动出勤退清打卡功能。
